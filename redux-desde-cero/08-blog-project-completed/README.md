# Blog Project

## Descripción

Código base del proyecto Blog Project donde se ven los conceptos basicos de redux y redux toolkit y RTK Query.

---

## Comandos

- npm i
- npm run dev
- npm run build
- json-server --watch data/db.json --port 3500

## Dependencias

- Axios
- Date Fns
- React
- React DOM
- React Icons
- React Redux
- React Router DOM
- Redux/toolkit
- Sweetalert2

## Dependencias De Desarrollo

- Vite
