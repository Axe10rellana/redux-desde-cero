# Redux Login Authentication

## Descripción

Código base del proyecto Redux Login Authentication donde se ven los conceptos basicos de redux, redux toolkit, RTK Query y Json Web Tokens.

---

## Comandos

- npm i
- npm run dev
- npm run build

## Dependencias

- React
- React DOM
- React Redux
- React Router DOM
- Redux/toolkit

## Dependencias De Desarrollo

- Vite
