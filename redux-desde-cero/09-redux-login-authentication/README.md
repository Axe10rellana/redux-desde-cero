# Redux Login Authentication

## Descripción

Código base de la API de registro, login y autenticación de usuarios donde se ven conceptos basicos de node, express, mongodb y Json Web Tokens.

---

## Comandos

- npm i
- npm start
- npm run dev
- npm run build

## Dependencias

- bcrypt
- cookie-parser
- cors
- date-fns
- dotenv
- express
- jsonwebtoken
- mongoose
- uuid

## Dependencias De Desarrollo

- nodemon
